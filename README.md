# API-Gathering
Code for extracting information from Contrataciones Públicas Ecuador API
In the process of building the DataHub we defined the data we would be the most interested in extracting. One of such data is the one for public contracts, provided by
Ecuador's government. A web scraping algorithm won't be necessary for this operation, because the government has designed an API that allows us to browse through the data
and aquire all the information available for each contracting project the government's institutions have concluded.

The following code shows makes the API call using the endpoint, and extracts a user determined set of observations, along with the variables specified by the user.
